## Summary

[Base](https://www.base.org/) is an Optimistic L2 rollup built by [Coinbase](https://www.coinbase.com/) on top of the OP Stack. As of [October 30th 2024](https://base.mirror.xyz/eOsedW4tm8MU5OhdGK107A9wsn-aU7MAb8f3edgX5Tk), Base **does** implement the OP Stack [fault proof system](https://docs.optimism.io/stack/protocol/fault-proofs/explainer) and as such verification of data is handled in the same manner as [Optimism](/chains/optimism).

- The Base explorer can be found [here](https://basescan.org).
- The addresses of the Base rollup related contracts can be found [here](https://docs.base.org/docs/base-contracts/).
