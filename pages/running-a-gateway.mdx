# Running a gateway

If you would like to run your own gateway, you are more than welcome to do so.

To run a gateway:

## Checkout

- Checkout the [repository](https://github.com/unruggable-labs/unruggable-gateways/tree/main).

```bash 
git clone https://github.com/unruggable-labs/unruggable-gateways.git
```

## Setup

- Run the following setup commands
```bash
foundryup
forge install
bun i
```

## Configuration

- Copy the `.env.example` template to `.env` and define the appropriate environment variables.
```bash
ALCHEMY_KEY
INFURA_KEY
ANKR_KEY
```

## Run it

* `bun run serve <chain> [port]`
	* Chains: `arb1` `base-testnet` `base` `blast` `celo-alfajores` `cyber` `fraxtal` `lineaV1` `linea` `mantle` `mode` `op` `opbnb` `polygon` `redstone` `reverse-op` `scroll` `self-eth` `self-holesky` `self-sepolia` `shape` `taiko` `zksync` `zora`
	* Default port: `8000`

The latest chain identifiers can be seen in the [source code](https://github.com/unruggable-labs/unruggable-gateways/blob/main/scripts/serve.ts).
